# **东南大学校园二手交易平台 - 项目描述**

## 📋 项目概述

**东南大学校园二手交易平台**是一个基于Web的校园内部二手商品交易系统，旨在为东南大学师生提供一个便捷、安全、可靠的校园内部物品流转平台。本项目严格遵循软件工程开发流程，采用Python+Flask+MySQL技术栈实现，涵盖了完整的电商交易核心功能。

## 🎯 项目目标

- **促进校园资源循环利用**：为教材、电子产品、生活用品等提供二次交易渠道
- **构建安全可信交易环境**：通过校园身份认证确保交易双方的真实性
- **提供便捷的交易体验**：从浏览、搜索到支付、配送的全流程线上化服务
- **培养实践开发能力**：作为软件工程课程实践项目，锻炼全栈开发技能

## ✨ 核心功能

### **用户系统**
- 校园身份注册与登录（支持@seu.edu.cn邮箱验证）
- 个人资料管理
- 交易历史记录

### **商品管理**
- 多维度商品发布（标题、描述、图片、价格、库存）
- 智能商品搜索（支持模糊匹配、分类筛选）
- 热门商品推荐算法

### **交易流程**
- 临时购物车系统（session-based存储）
- 在线模拟支付接口
- 校园内配送地址选择
- 交易评价与反馈

### **后台管理**
- 商品审核机制
- 用户行为监控
- 数据统计与分析

## 🛠️ 技术架构

### **前端技术**
- HTML5 + CSS3 + JavaScript (ES6+)
- 响应式设计，支持移动端访问
- RESTful API 接口调用

### **后端技术**
- **框架**：Python Flask
- **数据库**：MySQL 8.0+
- **会话管理**：Flask-Session
- **安全性**：密码哈希、SQL注入防护、CSRF保护

### **开发环境**
- **操作系统**：Ubuntu 20.04+（VMware虚拟环境）
- **版本控制**：Git
- **代码规范**：遵循PEP8 Python编码规范

## 🔒 安全特性

1. **用户认证**
   - 密码加盐哈希存储（bcrypt）
   - 会话令牌管理
   - 登录失败限制

2. **数据保护**
   - SQL参数化查询防注入
   - 敏感信息脱敏显示
   - 交易数据完整性校验

3. **交易安全**
   - 库存一致性检查（ACID事务）
   - 并发访问控制
   - 订单状态追踪

## 📈 数据库设计

系统采用规范化的数据库设计，包含：
- **Users表**：用户基本信息与认证
- **Items表**：商品发布与管理
- **Orders表**：订单处理与状态跟踪
- **Order_Items表**：订单明细关联
- **Categories表**：商品分类系统

## 🚀 部署方案

### **开发环境**
```bash
# 环境配置
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 数据库初始化
mysql -u root -p < database/schema.sql
# 或直接使用migrate数据库迁移
flask db init
flask db migrate
flask db upgrade
# 启动应用
flask run --host=0.0.0.0 --port=5000
```

### **生产环境建议**
- Nginx反向代理
- Gunicorn WSGI服务器
- MySQL主从复制
- 定期备份策略
